# ORYON PLATFORM

## About
ORYON PLATFORM is a platform for sim racers developed by SIMERIS RACING. Its main goal is to make SIMERIS gear more flexible, customizable and to help mastering using that gear. We are planning to also add other tools, like laptime progression tracking, not only for SIMERIS gear users.
